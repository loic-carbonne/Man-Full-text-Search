NAME


Openbox - standards compliant, fast, light-weight, extensible window manager. 
SYNOPSIS




gdm-control <option> 
DESCRIPTION




gdm-control lets you control gdm from within an X session. It enables you to change s-1GDMs0's behaviour for when you end the current sesson. For instance, you can tell s-1GDMs0 to reboot, and then immediately log out of the current session, and the computer will be rebooted. 
OPTIONS






\fB--help\fR 5 



Display some help and exit. 

\fB--none\fR 5 



Do nothing special when the current session ends. 

\fB--shutdown\fR 5 



Shutdown the computer when the current session ends. 

\fB--reboot\fR 5 



eboot the computer when the current session ends. 

\fB--suspend\fR 5 



Suspend the computer when the current session ends. 

\fB--switch-user\fR 5 



Log in as a new user (this works immediately).
