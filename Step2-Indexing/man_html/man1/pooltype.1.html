NAME

pooltype - display a WEB pool file

SYNOPSIS

pooltype pool_file_name

DESCRIPTION

This manual page is not meant to be exhaustive. The complete documentation for this version of *(TX can be found in the info file or manual Web2C: A TeX implementation .

The pooltype program translates a (program-oriented) *(WB string pool file to a (human-oriented) text file.

The pool_file_name must be complete; no adding of default extensions or path searching is done.
