The

utility provides virtual half-line and reverse line feed sequences for terminals without such capability, and on which overstriking is destructive. Half-line characters and underlining (changed to dashing `-') are placed on new lines in between the normal output lines.

The following options are available:

option is useful for sending output to the line printer when the output contains superscripts and subscripts which would otherwise be invisible.
