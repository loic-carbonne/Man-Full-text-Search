NAME


leafpad - GTK+ based simple text editor 
DESCRIPTION




leafpad is a simple s-1GTK+s0 based text editor. The user interface is similar to Windows(tm) notepad. It aims to be lighter than GEdit and KWrite and to be as useful as them. 

s-1WARNING:s0 The encoding feature is derived from built-in Glib's iconv feature (g_convert), which you can check with command f(CW*(C`iconv -l*(C'. Leafpad can use only the listed encoding. Any other encoding cause serious data loss. 
SYNOPSIS






  leafpad [options] [filename] 

OPTIONS






\fB--codeset=CODESET\fR 4 



Set codeset to open file. 

\fB--display=DISPLAY\fR 4 



X display to use. 

\fB--help\fR 4 



Dispaly the command line options. 

\fB--screen=SCREEN\fR 4 



X screen to use. 

\fB--sync\fR 4 



Make X calls synchronous. 

\fB--tab-width=WIDTH\fR 4 



Set tab width. 

\fB--version\fR 4 



Display version number.

None.
