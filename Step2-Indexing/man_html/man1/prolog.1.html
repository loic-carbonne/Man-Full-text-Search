NAME

GNU Prolog - Prolog compiler, includes CLP(FD).

SYNOPSIS

gprolog
gplc
hexgplc
pl2wam
wam2ma
ma2asm
fd2c

DESCRIPTION

This program does not have a useful manpage.

If you install the gprolog-doc package, you will find GNU Prolog documentation in the file /usr/share/doc/gprolog-doc/manual.pdf or you can read it in HTML format starting with the file /usr/share/doc/gprolog-doc/gprolog.html/index.html or at the URL http://localhost/doc/gprolog-doc/gprolog.html/
