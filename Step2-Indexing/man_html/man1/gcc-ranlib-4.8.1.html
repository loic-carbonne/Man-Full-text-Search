NAME

gcc-ranlib - a wrapper around ranlib adding the --plugin option

SYNOPSIS

gcc-ranlib [s-1OPTIONs0] ... [s-1ARGSs0...]

DESCRIPTION

gcc-ranlib is a wrapper around ranlib(1) adding the appropriate --plugin option for the GCC 4.8 compiler.

OPTIONS

See ranlib(1) for a list of options that ranlib understands.

SEE ALSO

ranlib(1)
