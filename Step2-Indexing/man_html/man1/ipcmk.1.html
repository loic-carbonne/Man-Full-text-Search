NAME

ipcmk - make various IPC resources

SYNOPSIS

ipcmk [options]

DESCRIPTION

ipcmk allows you to create shared memory segments, message queues, and semaphore arrays.

OPTIONS

Resources can be specified with these options:

-M, --shmem \fIsize\fRCreate a shared memory segment of size bytes.

-Q, --queueCreate a message queue.

-S, --semaphore \fInumber\fRCreate a semaphore array with number of elements.

Other options are:

-p, --mode \fImode\fRAccess permissions for the resource. Default is 0644.

-V, --version Display version information and exit.

-h, --help Display help text and exit.

SEE ALSO

ipcrm (1), ipcs (1)
