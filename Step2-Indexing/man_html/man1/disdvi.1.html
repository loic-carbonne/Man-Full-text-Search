NAME

disdvi - `disassemble' a (p)TeX or XeTeX DVI file

SYNOPSIS

disdvi


disdvi -x


disdvi -h

DESCRIPTION

disdvi interprets the contents of dvi_file or xdv_file printing the DVI commands and text in a human-readable ASCII format on the standard output.

SEE ALSO

dvi2tty (1)
