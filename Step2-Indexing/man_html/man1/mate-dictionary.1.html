NAME

mate-dictionary - Look up words on dictionaries

SYNOPSIS

mate-dictionary 

mate-dictionary  [options]

or select Dictionary from the Accessories submenu of the Applications menu.

DESCRIPTION

MATE Dictionary provides dictionary definitions of words, using a dictionary source.

For full documentation see the MATE Dictionary online help.

OPTIONS

--look-up word Looks up the specified word using the pre-defined dictionary source.

-s source or --source source Uses the specified source for looking up words. This does not affect the global settings.

-l or --list-sourcesLists all the sources available.

-n or --no-windowUsing this switch with the --look-up will print the definitions found on the console without launching the GUI.

--help Display help information.

CONFIGURATION

All the configuration is handled using Gsettings.
