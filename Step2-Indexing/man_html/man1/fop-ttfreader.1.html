NAME


fop-ttfreader --  A tool which reads TTF files and generates XML font metrics file for use in FOP. 
SYNOPSIS




fop-ttfreader [options] fontfile.ttf xmlfile.xml 
OPTIONS






\fB-enc\fR ansi 4 



with this option you can create a WinAnsi encoded font. The default is to create a s-1CIDs0 keyed font.  If you're not going to use characters outside the pdfencoding range (almost the same as iso-8889-1) you can add this option. 

\fB-ttcname\fR fontname 4 



if you're reading data from TrueType Collection (.ttc file) you must specify which font from the collection you will read metrics from. If you read from a .ttc file without this option, the fontnames will be listed for you. 

\fB-fn\fR fontname 4 



default is to use the fontname in the .ttf file, but you can override that name to make sure that the embedded font is used (if you're embedding fonts) instead of installed fonts when viewing documents with Acrobat Reader. 
EXAMPLES










generates freeserif.xml file containing s-1XMLs0 font metrics from FreeSerif.ttf
