NAME


rename - renames multiple files 
SYNOPSIS




rename [ -v ] [ -n ] [ -f ] perlexpr [ files ] 
DESCRIPTION




f(CW*(C`rename*(C' renames the filenames supplied according to the rule specified as the first argument. The perlexpr  argument is a Perl expression which is expected to modify the f(CW$_ string in Perl for at least some of the filenames specified. If a given filename is not modified by the expression, it will not be renamed. If no filenames are given on the command line, filenames will be read via standard input. 

For example, to rename all files matching f(CW*(C`*.bak*(C' to strip the extension, you might say 



        rename *(Aqs/e.bak$//*(Aq *.bak 



To translate uppercase names to lower, you'd use 



        rename *(Aqy/A-Z/a-z/*(Aq * 

OPTIONS






\fB-v\fR, \fB--verbose\fR 8 



Verbose: print names of files successfully renamed. 

\fB-n\fR, \fB--no-act\fR 8 



No Action: show what files would have been renamed. 

\fB-f\fR, \fB--force\fR 8 



Force: overwrite existing files. 
ENVIRONMENT




No environment variables are used.

Larry Wall

DIAGNOSTICS




If you give an invalid Perl expression you'll get a syntax error.
