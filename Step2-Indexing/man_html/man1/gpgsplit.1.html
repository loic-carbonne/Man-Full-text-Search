NAME

gpgsplit - Split an OpenPGP message into packets

SYNOPSIS

gpgsplit

DESCRIPTION

This manual page documents briefly the gpgsplit command.

gpgsplit splits an OpenPGP message into packets.

OPTIONS

-v, --verboseVerbose.

-p, --prefix \fISTRING\fRPrepend filenames with STRING.

--uncompress Uncompress a packet.

--secret-to-public Convert secret keys to public keys.

--no-split Write to stdout and don't actually split.
