Name

policytool - PolicyTool Administration GUI Utility

f3policytool reads and writes a plain text policy file based on user input via the utility GUI.

SYNOPSIS

Run the policytool Administrator's utility

f4policytool

Run policytool and load the specified policy file

f4policytoolf2[-file f2filenamef2]

where:

file directs f2policytool to load a local policy file

filename The file name

DESCRIPTION

f3policytool is a GUI that allows users to create and manage policy files. For details, see

f2the Policytool Users Guide @

http://docs.oracle.com/javase/7/docs/technotes/guides/security/PolicyGuide.html.

OPTIONS

file Loads f2filename.
