NAME

mkindex - script to process LaTeX index and glossary files

SYNOPSIS

mkindex BASENAME[.idx] ...

DESCRIPTION

mkindex runs makeindex(1) on any index (f(CW.idx) or glossary (f(CW.glo) files created by TeXing the file BASENAME.tex.

FILES

\fIgind\&.ist\fP

MakeIndex index style file

\fIgglo\&.ist\fP

MakeIndex glossary style file

SEE ALSO

makeindex(1)
