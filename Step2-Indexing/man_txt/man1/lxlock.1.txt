NAME
       lxlock - locking utility for LXDE

SYNOPSIS
       lxlock


DESCRIPTION
       lxlock is a simple script to lock the session, using third application.
       It currently those applications, in this order (try the next one if the
       application is not available) : light-locker xscreensaver gnome-screen-
       saver slock xlock i3lock
