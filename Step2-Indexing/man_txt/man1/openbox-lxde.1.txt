NAME
       openbox-lxde - wrapper to run openbox for LXDE

SYNOPSIS
       openbox-lxde

DESCRIPTION
       This manual page documents briefly the openbox-lxde command.

       openbox-lxde is a wrapper script which runs openbox with LXDE specific
       config file.
