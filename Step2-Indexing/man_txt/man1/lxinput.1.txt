NAME
       lxinput - a light-weight program to configure keyboard and mouse

SYNOPSIS
       lxinput

DESCRIPTION
       This manual page documents briefly the lxinput command.

       lxinput is a program to configure keyboard and mouse for desktop,
       usually for LXDE.
