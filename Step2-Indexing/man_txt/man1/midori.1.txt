NAME
       midori - fast and lightweight web browser


SYNOPSIS
       midori [OPTION...] [URL]


DESCRIPTION
       midori is a lightweight web browser.


OPTIONS
   Application Options:
       -v, --version
              Display program version

       --display=DISPLAY
              X display to use


   Help Options:
       -?, --help
              Show help options

       --help-all
              Show all help options

       --help-gtk
              Show GTK+ Options
