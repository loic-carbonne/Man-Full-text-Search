NAME
       gummi - LaTeX editor

DESCRIPTION
       gummi is a simple LaTeX editor for GTK+.

OPTIONS
       -d     run gummi with debug logging enabled.

       -h     display command line options.

BUGS
       Undoubtedly, please report at: http://gummi.midnightcoding.org
