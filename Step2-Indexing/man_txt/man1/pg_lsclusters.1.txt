NAME
       pg_lsclusters - show information about all PostgreSQL clusters

SYNOPSIS
       pg_lsclusters [options]

DESCRIPTION
       This command shows a list about the configuration and status of all
       clusters.

OPTIONS
       -h, --no-header
           Do not print the column header line.
